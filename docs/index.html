Air Script Glove

Project Type: AI + Embedded System

This project converts hand-written air gestures into text using an MPU6050 sensor mounted on a glove. The system uses ESP32 for processing and Edge Impulse for gesture recognition.

Components Used:

  • ESP32 DevKit V1
  • MPU6050 Sensor
  • OLED Display
  • TP4056 Charging Module
  • Push Button

Features:

  • Air gesture recognition (A, C, E, I, V)
  • Special gestures for Space, Enter, Backspace
  • Bluetooth text transmission
  • OLED display feedback

Application:

Designed for motor-impaired individuals to enable communication through air writing.
