✋ Air Script Glove

Gesture to Text Conversion System

📌 Project Overview

Air Script Glove is an AI-based embedded system that converts air-written gestures into text. It is designed to help motor-impaired individuals communicate easily without using keyboards.

⚙️ System Architecture

  • MPU6050 captures motion data
  • ESP32 processes the data
  • Edge Impulse performs gesture recognition
  • OLED displays output
  • Bluetooth transmits text

🔩 Components

ESP32
MPU6050
OLED
TP4056
Push Button

🔄 Working

  1. User writes in air
  2. Sensor captures motion
  3. AI model classifies gesture
  4. Output shown on OLED
  5. Text sent via Bluetooth

✨ Features

  • Recognizes A, C, E, I, V
  • Space → Right swipe
  • Backspace → Left swipe
  • Enter → Upward motion
  • Real-time display

🎯 Applications

  • Assistive communication
  • Wearable computing
  • Hands-free typing

🚧 Status

Currently in development phase with limited gesture set.

🔮 Future Scope

  • Full alphabet recognition
  • Mobile app integration
  • Improved accuracy
